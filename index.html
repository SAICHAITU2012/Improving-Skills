Detailed phase-by-phase progression from foundation to job readiness

Phase 1: Foundation & Language (Month 1-2)

Goal: Speak AI fluently and write functional Python

Foundation 8 weeks
  • Complete "AI For Everyone" (Andrew Ng)

    Non-technical foundation understanding AI/ML/GenAI concepts

    6 hours Coursera
  • Finish first 4 courses of "Python for Everybody"

    Build Python proficiency with practical, beginner-friendly approach

    20 hours Coursera
  • Build 5 working Python scripts automating medical coding tasks

    15 hours Hands-on
  • Complete basic Pandas and data analysis exercises

    Essential data manipulation skills for AI/ML workflows

    10 hours Kaggle

Phase 2: Core AI Mechanics (Month 3-4)

Goal: Build strong ML foundations and transition smoothly into generative AI

Core Skills 8 weeks
  • Machine Learning Specialization — Andrew Ng

    Gold-standard course for understanding ML intuition and model mechanics

    60 hours Coursera
  • Kaggle Learn Micro-Courses

    Hands-on practice on real datasets: ML, deep learning, data cleaning

    15 hours Kaggle
  • Build Your First ML Project — Healthcare Focus

    Apply ML to a practical medical dataset for portfolio-ready experience

    20 hours Project
  • Generative AI with LLMs — Coursera

    Bridges ML and LLMs; covers transformers, embeddings, and fine-tuning

    16 hours Coursera
  • Prompt Engineering for Developers — OpenAI / Coursera

    Master structured prompt patterns and use GPT-4 efficiently

    2 hours Coursera
  • Subscribe to ChatGPT Plus

    Use GPT-4, DALL-E, and advanced data analysis as your interactive lab

    Ongoing Tool
  • Build First LLM Mini-Project

    Create a small generative-AI tool — e.g., medical-chatbot prototype

    20 hours Project

Goal: Deep mastery of LLMs, creative AI tools, and real-world application design

Advanced 4 weeks

  • Prompt Engineering Specialization — Vanderbilt

    Example Implementation: Medical Documentation Assistant that summarizes reports, suggests ICD-10 codes, and generates patient education materials using multimodal AI

    See detailed specifications in Portfolio Projects section

    40 hours Capstone Project Portfolio Centerpiece
